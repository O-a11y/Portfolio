Scratch Project: Arctic Platformer

During September 2019, I collaborated with Rohan Kanhai on a Scratch Project for a Computer Science course we are enrolled in. We coded a platforming game, which features several levels and a shrinkable ball.


To make this game work, we had to write the code for the controls (WASD) as well as create a working gravity. The 'S' control is special in that it makes the player's ball shrink vertically to fit through small spaces. Next, we created different varieties of obstacles and ways to overcome them. Finally, there is a challenge level at the end that is significantly more difficult than the preceeding ones.
